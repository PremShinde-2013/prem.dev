## Installation

### Clone the Repository

<CodeGroup>

```bash {title:git}
git clone https://github.com/yourusername/cryptoversx.git
```

</CodeGroup>

### Navigate to the Project Directory

<CodeGroup>
````bash {title:git}
cd cryptoversx
````
</CodeGroup>

### Install Dependencies

<CodeGroup>

```bash {title:npm}
npm install
```

```bash {title:yarn}
yarn install
```

```bash {title:pnpm}
pnpm install
```

</CodeGroup>
### Run the Development Server

<CodeGroup>

```bash {title:npm}
npm run dev
```

```bash {title:yarn}
yarn run dev
```

```bash {title:pnpm}
pnpm run dev
```

</CodeGroup>
## Build for production:

<CodeGroup>

```bash {title:npm}
npm run build
```

```bash {title:yarn}
yarn run build
```

```bash {title:pnpm}
pnpm run build
```

</CodeGroup>

## Start the production server:

<CodeGroup>

```bash {title:npm}
npm start
```

```bash {title:yarn}
yarn start
```

```bash {title:pnpm}
pnpm start
```

</CodeGroup>
